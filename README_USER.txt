===============================================
autOTP - OTP 자동 로그인 프로그램
버전: 2.0
===============================================

【설치 방법】

1. ZIP 파일 압축 해제
2. autOTP.exe 더블클릭으로 실행

※ Python 설치 필요 없음!


【Windows Defender 차단 시】

프로그램이 처음 실행될 때 Windows Defender가
"알 수 없는 게시자"라는 경고를 표시할 수 있습니다.

✅ 해결 방법:
  1. "추가 정보" 클릭
  2. "실행" 버튼 클릭

또는 수동으로 제외 항목 추가:
  1. Windows 보안 열기
  2. 바이러스 및 위협 방지 > 설정 관리
  3. 제외 항목 > 제외 항목 추가
  4. 파일 선택 후 autOTP.exe 추가


【첫 실행 시 주의사항】

⏳ 첫 실행 시 EasyOCR 모델을 다운로드합니다.
   - 인터넷 연결 필요
   - 5~10분 소요 (약 500MB 다운로드)
   - 프로그램이 멈춘 것처럼 보일 수 있으나 정상입니다

✅ 두 번째 실행부터는 빠르게 시작됩니다!

*가끔 계정을 입력하지 않는 버그가 있습니다. 
 그때는 자동로그인을 중단하고 학번 입력창을 한번 클릭해주세요.


【사용 방법】

1. 프로그램 실행
2. 학교 계정(학번)과 비밀번호 입력
3. "설정 저장" 클릭
4. 선택사항: "윈도우 시작시 자동 실행" 체크
5. 학교 통합로그인 창이 뜨면 자동으로 OTP 입력됩니다


【사용 중 주의사항】

⚠️  OTP 입력 중에는 마우스를 움직이지 마세요
⚠️  ISign+ OTP 앱이 작업표시줄에 있어야 합니다
⚠️  인터넷 연결 필요 (OTP 인증)
⚠️  만약 파일 위치를 옮기고 싶으시다면 exe 파일만       
    옮기는 것이 아닌 파일 통째로 옮기셔야합니다!!


【문제 해결】

Q. 프로그램이 실행되지 않아요
A. Windows Defender 제외 항목에 추가하세요 또는
   잠시 기다려주세요

Q. OTP 인식이 안돼요
A. 간혹 해상도 문제 또는 어플 클릭이 실행되지 않았을 때
   발생 할수도 있습니다. 해상도를 1920x1080으로 수정 후 전체화면으로 한 프로그램을 최소화 해주세요.

Q. 계정 정보를 삭제하고 싶어요
A. 프로그램 내 "계정 완전히 삭제" 버튼 클릭

Q. 첫 실행이 너무 오래 걸려요
A. EasyOCR 모델 다운로드 중입니다 (최초 1회만)
   인터넷 연결 확인 후 기다려주세요

Q. 로그인을 중단하고 싶어요
A. 로그인 진행 창에서 "로그인 중단" 버튼 클릭


【안전성】

✅ 모든 계정 정보는 Windows 자격 증명 관리자에
   암호화되어 저장됩니다

✅ 외부 서버로 데이터를 전송하지 않습니다
   (학교 서버 로그인 제외)

✅ 오픈소스 라이브러리만 사용합니다
   - PyQt5 (UI)
   - pywinauto (윈도우 제어)
   - EasyOCR (문자 인식)
   - keyring (계정 정보 암호화)


【시스템 요구사항】

- Windows 10/11
- 인터넷 연결
- ISign+ OTP 앱 설치 필요
- Mac/iPhone 환경 지원 불가
